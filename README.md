## 🔒 Firebase Resume Management System Setup

This repository contains a secure resume management feature. To set it up on your own deployed copy:

1. **Authentication**: Enable **Email/Password** provider in Firebase Authentication and create your admin account.
2. **Cloud Firestore**: Initialize a database and paste the rules from the `walkthrough.md` to secure writing.
3. **Configuration**: Replace the config keys inside [index.html](file:///a:/Portfolio/files%20(1)/index.html) with your own credentials.
4. **Login**: Double-click the avatar image or press `Ctrl + Shift + A` to show the secure login modal and start uploading!
